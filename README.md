# dlproxy - 通用文件下载加速代理

一个基于 Go 语言开发的高性能、低能耗、支持工业级安全防护的大文件反向代理中转加速服务。内置 SSRF 防御、DNS Rebinding 拦截，支持多线程工具（如 IDM、迅雷）以及终端（wget、curl）的断点续传与 Range 分块下载。

---

## 📋 环境要求

* **Go 语言版本**：`>= 1.26.4` (推荐使用官方最新稳定版) [💡]

---

## 🚀 快速开始与部署指南

请按照以下简单步骤，在您的本地或公网服务器上完成编译与运行：

### 1. 克隆或下载源码
首先，新建一个文件夹，并直接拉取本仓库的完整源码：
```bash
git clone https://github.com/lemonsn/dlproxy
```
> **💡 提示**：您也可以选择不克隆仓库，直接在新建的文件夹内，单独下载本仓库中的 `main.go` 和 `template.go` 两个核心文件。

### 2. ⚙️ 自定义端口（可选）
服务的默认监听端口为 `8080`。如果您需要更换其他端口（例如改为 `9090`）：
* 请打开 `main.go`，找到 `address := ":8080"` 这一行，直接将 `8080` 修改为您期望的端口即可。*(注：前端使用的是相对路径表单，无需进行任何端口修改)*

### 3. 初始化 Go 模块
进入包含源码的文件夹内，初始化 Go 语言的包管理器（模块名已被定义为 `dlproxy`）：
```bash
go mod init dlproxy
```

### 4. 编译程序
使用 Go 编译器将源码打包编译为原生的、无依赖的独立可执行文件：
```bash
go build
```

### 5. 运行服务
编译完成后，直接运行生成的可执行文件即可启动中转代理服务：
* **Linux / macOS:**
  ```bash
  ./dlproxy
  ```
* **Windows:**
  ```cmd
  dlproxy.exe
  ```

---

## 🖥️ 客户端使用方式

### 🌐 浏览器/网页端
服务启动后，在浏览器中直接访问 `http://localhost:8080`（若修改过端口，请访问相应的新端口）。在精美的 H5 脱敏前端面板中，粘贴您想下载的任意大文件链接，并输入本站的专属通关密码（默认密码为 `admin123`），点击按钮即可全速拉取。

### 🐚 终端自动化下载 (不改动脚本直接使用)
本仓库中额外附带了终端自动下载脚本 `dl.sh`。当您在 Linux/macOS 的终端中需要加速下载时，赋予执行权限后直接运行即可（脚本会自动为您嗅探系统中可用的 `wget`、`curl` 或 `aria2c` 工具并自动调用）：

```bash
chmod +x dl.sh
./dl.sh "您的目标大文件下载链接" [可选密码] [可选服务器地址]
```
